What is github actions ?
    This is a cicd platform  provided by GitHub.


Advantages of Actions

    - Fully integrated with GitHub
    - Respond to any GitHub event
    - Community-powered workflows
    - Any platform, any language, any cloud



Components of an Action

Events  > Workflows > Actions

Events
    GitHub triggered events like push, pull request etc

    It can also be on a schedule

Workflows
    These are like pipelines codes, however this is in yaml
